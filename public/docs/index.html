OnePad API Document

Create API 新規作成API

Request

POST http://onepad.anone.me/create.json
name value
content 本文

Response

status
201 Created 成功
400 Bad Request 失敗。contentがない。
409 Conflict 失敗。ランダムに生成したkeyが空いていなかった。
413 Request Entity Too Large 失敗。contentが長すぎる。
500 Internal Server Error 失敗
503 Service Unavailable 失敗。短時間の間に新規作成されすぎている。

name value
key key
key key

Response

status
